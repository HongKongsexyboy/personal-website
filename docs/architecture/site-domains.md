# Site Domains

## 1. Home

- 展示个人定位、核心优势、精选项目与主要 CTA。

## 2. Projects

- 支持项目列表、标签筛选、详情页、GitHub、演示地址、技术栈和难点复盘。

## 3. Blog / Notes

- 承载技术文章、Unity 学习记录、面试总结和渲染/架构专题。

## 4. About

- 承载个人简介、经验、技能图谱和时间线。

## 5. Contact

- 提供联系表单、邮箱、社交链接和简历下载。

## 6. SEO

- 动态 metadata、sitemap、robots、Open Graph 和 RSS。

## 7. Analytics

- 预留 PV、UV、停留时长、项目点击和简历下载转化接入点。

## 8. Platform

- 预留日志、错误监控、环境变量、部署流水线与安全头。
